Projeto de estudo prático de **React 19** desenvolvido para consolidar conceitos fundamentais e intermediários do framework. Consome a **API do GitHub** para exibir perfis e repositórios de usuários.

A hands-on **React 19** study project built to consolidate fundamental and intermediate concepts. Consumes the **GitHub API** to display user profiles and repositories.

---

## Como Executar / Getting Started

```bash
# Instalar dependências / Install dependencies
npm install

# Iniciar JSON Server (mock) / Start JSON Server (mock)
npm run json-server

# Em outro terminal / In another terminal
npm run dev
```

O projeto estará disponível em **http://localhost:5173**.
